## Copy Undertale music files from the original game here (*.obb)

### -- Windows --

**OPEN CMD HERE** (`undertale-tui/assets/music`)

* **If Steam**, paste:
    ```cmd
    copy "C:\Program Files (x86)\Steam\steamapps\common\Undertale\*.obb" .
    ```

* **If GOG**, paste:
    ```cmd
    copy "C:\GOG Games\Undertale\*.obb" .
    ```

* **If GOG Galaxy**, paste:
    ```cmd
    copy "C:\Program Files (x86)\GOG Galaxy\Games\Undertale\*.obb" .
    ```
